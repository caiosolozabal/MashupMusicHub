## 🚀 Como colocar no ar (mashupmusic.com.br)

1. **Subir para o GitHub**: Garanta que o código esteja em um repositório privado.
2. **Conectar ao Firebase App Hosting**: No console do Firebase, selecione "App Hosting" e conecte seu repositório. O Firebase cuidará do build automático do Next.js.
3. **Apontar Domínio**:
   - No console do Firebase, adicione o domínio `mashupmusic.com.br`.
   - Copie as entradas DNS (A e TXT).
   - Insira-as no seu painel do **Registro.br**.
4. **Logotipo**: Para exibir sua marca, coloque o arquivo `logo.png` na pasta `/public/`.
